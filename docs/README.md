## 💻 기능 목록

- 구입 금액을 입력 받기

  - 에러 발생
    - 숫자가 아닌 경우
    - 소수점이 있는 경우

- 해당 구입 금액으로 살 수 있는 로또 수량 구하기

- MissionUtils 라이브러리 사용하여 발행한 로또 번호 출력하기

  - 에러 발생
    - 숫자가 아닌 경우
    - 6자리가 아닌 경우
    - 1~45 정수 아닐 경우

- 당첨 번호 입력받기

  - 에러 발생
    - 숫자가 아닌 경우
    - 6자리가 아닌 경우
    - 1~45 정수 아닐 경우

- 보너스 번호 입력받기

  - 에러 발생
    - 1~45 정수 아닐 경우

- 당첨 내역 출력
